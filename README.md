# DANA Mutasi Bot

Bot Telegram untuk otomatisasi pencatatan transaksi DANA ke Google Sheets.

## Setup
1. Copy semua file ke folder
2. Deploy ke Render.com
3. Setup environment variables
4. Share Google Sheets ke service account

## Usage
- `/start` - Start bot
- `/list` - List accounts
- `/tambah` - Add account
- `/stop` - Stop account
